# 🚜 AgriShield AI: Climate Risk & Farm Decision Agent

AgriShield AI is a modern multi-agent decision support system designed to help farmers mitigate climate risks, predict plant pathologies, and track crop market valuations.

## 🚀 Dynamic Features
* **Multi-Agent Decision Orchestrator**: Coordinates specialized AI sub-agents (Climatology, Agronomy, Pathology, Trading, and Resource Engineering) using Gemini-2.5-Flash.
* **Local Data Grounding**: Prevents hallucination by matching coordinates against regional CSV telemetry layers.
* **Premium Dark Mode**: Built with glassmorphic cards and responsive metrics using Streamlit.
